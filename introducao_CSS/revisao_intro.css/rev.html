Revisão/ introd. CSS

Seletor ID

É unico, só pode ser usado apenas uma vez numa pag. web

Propriedade Font

Responsavel pela formatação da fonte. Principais: font-size; font-family; font-style; font-height.

Propriedade TEXT

Muda as caracteristicas do bloco de texto. Principais: text-align; text-decoration; text-indent; tex-transform.

Box Model

É um modelo de caixa, que contem bordas, largura, altura etc, só que não visivel sem background... assim como uma caixa.

Padding

Adiciona em todos os lados

Margin

adiciona distância da borda para fora em todos os lados.
Outro exemplo: margin-left: auto; margin-right: auto; ele pega os espaços que sobram quando se formata somente com height.

Bordas

Adiciona bordas aos blocos

Outline

Adiciona uma borda que fica para fora do bloco de linha

Propriedade Display

Pode adicionar blocos em linhas no paragrafo sem a quebra de linha, com quebra de linha etc

Pseudo-clsses

Serve para fazer a formatação não dependendo de uma classe. Usado somente no "link"

cachorro quente Só de osasco.

List-Style

Serve para formatar os bolets, seja em, listas-ordenada ou não ordenadas
