Unidades de medidas

Pixels: Mais utilizado entre os iniciantes, mas devemos ter cuidado ao usar, pois, não é tão profissional. Ex. de onde podemos usar: numa margem pequena, box-shadow, text-shadow.

Em: Equivale a 16 pixels e para adicionar mais pixels basta colocar ponto(.) e número (0215), se adicionar no CSS uma porcentagem em font-size (62.5%) HTML. Ex: 1.541em.
A unidade de medida (em) respeita sempre a fonte de elemento PAI.

Rem: Diferente do em, ele respeita o root (nesse caso a fonte no html). Então é necessário tomar cuidado com qualquer alteração feita no root (font-size no HTML), pois, séra reflatida no rem.
