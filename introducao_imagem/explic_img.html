Formas de inserir Imagens

Html: usa-se a tag (img src="..." alt="..."). Ela requer a tag "search(origem da imagem)" e o atributo "alt(é usado um texto descritivo sobre o que a imagem está representando)".

CSS: usa-se a cod. (background-image: url(...);).

Qual é a melhor? bom... se html é conteudo, deve se usar para representar o texto que está ali. Já o CSS, se a imagem for decorativa deve ser usado para ilustrar a parte individual.


ALERTA!! Mesmo que não tenha um texto descritivo para tal imagigem é melhor incluir um texto vazio(ALT")

Imagem vetorial X Imagem rasterizada

Imagem rasterizada: é uma imagem bitmap(comuns para fotografias), ou seja, formada por pixels, onde cada pixels vai ter uma determinada cor, gerando uma imagem.


Imagem vetorial: Uma img vetorialé criada com base em calculos matematicos(posição x,y etc). Tende a ser mais leves pois, é mmais facil representar com uma imagem do que com pixels. Usada em poucas cores e poucos elementos, não perde a resoução

    Tipos de imagens:
  • .jpg/.jpeg- imagem bitmap, arq menores em termos de kbytes
  • .png- tende a ter um tamanho menor, pode ter uma transparencia
  • .gif- é uma imagem leve com ou sem transparencia
  • svg- formato vetorial, idelal para logotipo
  • .webp- é um formato novo, que utiliza menos kbytes e a taxa de compressão é maior sem perder qualidade de imagem. Não é muito utilizado hoje em dia por não ter browsers.
  • .avif- é um formato novo, que utiliza menos kbytes e a taxa de compressão é maior sem perder qualidade de imagem. Não é muito utilizado hoje em dia por não ter browsers.

    • caniuse usado para dizer qual tecnologias posso usar dependendo do suporte que tiver dos browsers.
